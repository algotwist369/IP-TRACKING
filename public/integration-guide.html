Step 1: Update Your Server URL

Edit the tracking script and change the server URL:

// In tracking-script.js, line 8: const TRACKING_SERVER = 'http://your-domain.com:5000'; // or for production: const TRACKING_SERVER = 'https://your-domain.com';

Step 2: Include the Script in Your Website

Add this script tag to your HTML pages (preferably before closing </body> tag):

<script src="http://your-domain.com/tracking-script.js"></script>
Important: Make sure the script is accessible from your website. You may need to:
  • Upload the script to your web server
  • Use a CDN
  • Host it on the same domain as your tracking server

Step 3: Test the Integration

Visit your website and check the browser console for tracking messages:

// Expected console output: "Tracking visit: {website: 'example.com', referer: 'https://google.com', ...}" "Visit tracked successfully: Visit tracked" "Session ID: abc123..." "Visit type: external"

Step 4: Verify in Dashboard

Check your tracking dashboard to see the new visit data with session information:

// New fields in your dashboard: - sessionId: Unique session identifier - visitType: 'external', 'direct', or 'internal' - isFirstVisit: true/false - referer: Source of the visit

Advanced Configuration

Custom Session Duration:

// In tracking-script.js, line 9: const SESSION_DURATION = 60 * 60 * 1000; // 1 hour instead of 30 minutes

Manual Tracking:

// Force track a visit window.IPTracker.track(); // Check if should track if (window.IPTracker.shouldTrack()) { // Custom logic here window.IPTracker.track(); } // Get session info const session = window.IPTracker.getSession(); console.log('Current session:', session);

Custom Data Collection:

// Get all tracking data const data = window.IPTracker.collectData(); console.log('Tracking data:', data);
