üìã Implementation Steps

1. Basic Implementation

Add this code to your HTML <head> section:

<!-- IP Tracker Configuration --> <script> window.IPTrackerConfig = { trackingCode: 'ip_your_user_id_timestamp_random', website: 'yourdomain.com', enableRealTime: true, enableVPNDetection: true, enableDeviceFingerprinting: true, enableBehavioralTracking: true, privacyMode: 'standard' }; </script> <!-- IP Tracker Script --> <script src="https://yourdomain.com/tracking-script.js"></script>

2. Configuration Options

  • trackingCode: Your unique tracking code from the dashboard
  • website: Your website domain
  • enableRealTime: Enable real-time tracking and heartbeats
  • enableVPNDetection: Enable VPN and proxy detection
  • enableDeviceFingerprinting: Enable device fingerprinting
  • enableBehavioralTracking: Track user behavior and interactions
  • privacyMode: 'standard', 'enhanced', or 'minimal'

3. Advanced Usage

You can also use the IP Tracker API programmatically:

// Track custom events IPTracker.trackEvent('button_click', { buttonId: 'signup', page: 'homepage' }); // Get current tracking data const data = IPTracker.getData(); console.log('Current tracking data:', data); // Manually track something IPTracker.track({ customField: 'customValue' });
